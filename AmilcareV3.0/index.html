Installation Instructions

  1. Create the destination directory where Amilcare will be installed
  2. Download and unpack the provided zip file
  3. run the bat file called Amilcare.bat

Caveat

  • The manual has not been updated for version 3.0, it is still that of V2.4. For example there is no longer an installation wizard. We will soon provide an updated version of the manual.

Known Problems

If you receive a Gate error during initialization about some V8 problems, try to download the following file, unzip it and put it in your JAVA_HOME/lib. If that file is already existent, SAVE YOUR COPY and replace it.

  • tools.jar
