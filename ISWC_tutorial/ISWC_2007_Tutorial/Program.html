The tutorial will last 1⁄2 day; the presentation style will be via slides and some demos (via videos) of real world applications. 


The tutorial will cover the following topics:


• Introduction and scene setting (10 minutes) 

• Issues in knowledge management in large organisations (20 minutes)

    o issues, size, media

    o requirements 

    o examples drawn from real use cases derived from the projects

IST X-Media and DTI-IPAS

 

Semantic web technologies: ontologies (30 minutes)

• use and role of ontologies: motivations and issues

• cost of ontologies

• scale of ontologies


Semantic web technologies for knowledge acquisition (45 minutes)

• issues in knowledge acquisition: 

• acquiring: what and what for?

• user centred methodologies and tools for text and image annotation

• automatic methodologies and tools for text annotation


Semantic web technologies for knowledge sharing, reuse and retrieval (45 minutes)

• issues in knowledge sharing

• approaches and novel methods to searching, sharing and reuse knowledge 


Conclusion and future work (15 minutes)

• unresolved issues

• the road ahead


Discussion (15 minutes)
