Assignment FAQs

Intelligent Web Module  

 
 

----- first part of the assignment -----

Q: The assignment does not specify if it is an individual or group assignment

A: An update has being sent out. It is a group assignment. Size of group must be <=3.


Q: Will I be marked differently if I do the assignment on my own.

A: No. You are discouraged from doing the assignment on your own. If you do so it is your choice.


Q: Can we use JQuery in our JavaScript code?

A: yes


Q: can we use GSON in our solution?

A: yes you can use anything that has been described and/or used in the lecture slides and lab classes.


Q: Can we use JSP instead of PHP for the assignment?

A: yes no problem. Usual rules: we must be able to open your zip file and run it on the personal Tomcat server

it is your responsibility to provide detailed instruction and any library necessary to run your solution


Q: can I use any Javscript library that helps with rendering (e.g. templates like Mustache.js or JsRender.js) or some special css libraries (e.g. compass+sass)? 

A: yes in general BUT make sure that you 1) check with me before using them so to make sure that you can 2) mention explicitly in the report that you used it and that you got permission from me and 3) any external library is included in your submission. We must not be required to download any software in order to run your solution.


Q: can I submit a jar file containing my Java solution?

A: You must. But make sure that you also submit the source code because we will use the source code. We will recompile the code. And make sure to submit the part of the assignment that is not java based!


Q: my personal Tomcat does not seem to work. How do I check that it actually works?

A: To check that the personal tomcat works, first of all check if you can load the examples in the directory “examples”. The best way to start implementing something is to copy the files from example into your context directory and then modify them introducing your own servlets and html/js files.


Q: I seem to have a problem with my Java Servlet on the departmental personal Tomcat. When I introduce the mapping in my web.xml file, my personal tomcat server stops working. Even html files that initially were loaded now are no longer loaded.

A: make sure to be using the same version of java as required by the server. Check the the tomcat log (the catalina.log file) for errors.


Q: If someone queries about a user (or keyword) which is already in the database, should the application search for new tweets of that user?

That is not specified in the document. It is up to you to find a reasonable strategy and to justify it in the report. I would personally think reasonable to ask the user to decide, but that is just a personal preference 


Q: should the form on QueryTwitter.html have fields for database access and Twitter tokens?

The way you pass the Twitter AOth is really up to you. As for me I just would like a reasonable solution. If you ask to set this for every query from the interface, or if you have the concept of sessions where you ask the user once for all, or if you have a preference file is up to you. I just need that this information is somehow declarative and I do not have to modify your code to change it


Q: What is the meaning of the following sentence in the assignment text:

                [5% of the code marks] - Code organization (e.g. use of Java packages – dcs.shef.ac.uk.com4280.net)

A: With regards to the use of Java packages, it is very important that you organise your code properly as it is not only part of the recommended Java coding standards, but it also makes our life easier when marking your code. Each package should basically contain groups of related types. So for instance, all your graphical interfaces could perhaps go inside a dcs.shef.ac.uk.com4280.gui package. In any case, if you still have problems with this, you should read about the use of packages here:

                http://docs.oracle.com/javase/tutorial/java/package/packages.html


Q: How long should the report be?

We think that it should be max 4 pages for the first part (and it will be 6 pages for the second part of the assignment you will receive soon). Please consider carefully the content of your report, as it carries quite a large chunk of the marks. In principle you should describe things in the clearest and most  concise way possible. Clear so that it passes the complete message of your design choices and in general decisions. Concise because it is important to avoid cluttering the report with lots of unuseful details. 6 pages are far better than 12,  if they are clear and contain the same amount of relevant information.


Q: can I change the names of output files, names of servlet, name of URL-Pattern?

A: No. if you changed them, our automated programs that check the results of the Intelligent Web assignment would no longer work. That would require us to adapt the test programs to every group's liking. We would never end in two weeks' time and would be a nightmare (misunderstandings, errors, etc.)

If you leave them as required, we will just run our test programs over all the assignments without stopping and we will get the marking of the result correctness done automatically (or nearly so). 

So no, you cannot change them. If you do you will be marked down: following instruction is a fundamental requirement for a computer scientist.
