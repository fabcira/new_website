Research Activities


Algorithms, Architectures and Applications of (Adaptive) Information Extraction from Text.
In the last years I have been focusing on the use of Adaptive IE for Knowledge Management.

Current Activities:

  • I am Sheffield's manager for the AKT project
  • Amilcare: a system for adaptive IE specifically designed for knowledge management
  • (LP)2 algorithm : on which both Amilcare and LearningPinocchio are based
  • I co-chair the NLP TAO discussion group

Former Activities:

  • LearningPinocchio: a system for adaptive Information Extraction from text used in a number of real world applications
  • The Pinocchio Toolkit: a classic IE system requiring limited knowledge of Natural Language Processing for building new applications
  • The ESTATE Project: the IE project at ITC-Irst I coordinated for a number of years
